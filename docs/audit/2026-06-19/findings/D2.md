## F-D2-008

- title: orchestrator_parse._matches_schema is unbounded recursion (stack DoS at depth ~1k)
- file: skills/bmad-story-automator/src/story_automator/commands/orchestrator_parse.py:103-110
- excerpt: |
    def _matches_schema(payload: object, schema: object) -> bool:
        if isinstance(schema, dict):

                    return False
- repro: |

- proposed_fix: Schema is read from a trusted local file, but payload comes from the LLM. Even if the schema is shallow (â‰¤2 levels), `json.loads` itself caps depth at recursion limit, so an attacker JSON with 1000 levels of `{"k":{"k":...}}` causes orchestrator process exit. Cap depth explicitly (`if depth > 16: return False`) inside `_matches_schema`. Bug 6-ish (DoS via depth).
